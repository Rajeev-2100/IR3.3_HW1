# My Food Delivery Website

A responsive Food Delivery Website built using React JS and React Router.  
This project allows users to explore restaurants, navigate between pages, and view detailed restaurant information with a clean and user-friendly interface.

## Quick Start

```bash
git clone https://github.com/Rajeev-2100/IR3.3_HW1.git
cd IR3.3_HW1
npm install
npm run dev
```

---

## Technologies Used

- React JS
- React Router
- Bootstrap
- Vercel (Deployment)

## Features

### Home Page
- Displays all restaurants
- Responsive and clean UI
- Easy navigation

### About Page
- Provides information about the website
- Explains restaurant delivery services

### Restaurant Page
- Displays restaurants in a structured layout
- Includes “View Details” button for each restaurant

### Restaurant Detail Page
Displays:
- Restaurant Name
- Cuisine Type
- Ratings
- Location
- Description
- Menu Items

### Responsive Design
- Optimized for mobile devices
- Tablet-friendly layout
- Desktop responsive interface

---

## Routes

| Route | Description |
|------|-------------|
| `/` | Home Page |
| `/about` | About Page |
| `/restaurants` | Restaurant Listing Page |
| `/restaurants/:id` | Restaurant Detail Page |

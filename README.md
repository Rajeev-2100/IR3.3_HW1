# My Food Delivery Website

A responsive Food Delivery Website built using React JS and React Router.
This project allows users to explore restaurants, navigate between pages, and view detailed restaurant information with a clean and user-friendly interface.

---

## Demo Link

[Live Demo](https://ir-3-3-hw-1.vercel.app/)

---

## Quick Start

```
git clone https://github.com/Rajeev-2100/IR3.3_HW1.git
cd IR3.3_HW1.git
npm install
npm run dev      # or `npm start` / `yarn dev`
```

## Technologies Used
- React JS
- React Router
- Bootstrap

## Features
**Home**
- Display a list of all restaurants
- Clean and responsive UI design

**About**
- Provides information about the website and restaurant services

**Restaurant Page**
- Shows all restaurants in a structured layout
- Includes a “View Details” button for navigating to the restaurant detail page

**Restaurant Detail Page**

Displays detailed restaurant information including:
- Restaurant Name
- Cuisine Type
- Location
- Ratings
- Short Description
- Menu Items
  
**Responsive Design**
Optimized for mobile, tablet, and desktop screens
